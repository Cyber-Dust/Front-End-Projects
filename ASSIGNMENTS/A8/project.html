Dasean Volk: Project Proposal

Project Name: COINSPACE

VISION

TOPIC: This webiste utilizes Coingecko API to query and display real time crypto data in a clean and smooth UX design.

The user is able to make a watchlist, and remove and add as many times as they want.

Additionally there is a News feed on the hottest crypto news, alongside an informational page about blockchain tech and theory.

AUDIENCE: Those interested in monitoring and analyzing real time cryptocurrency data.

DESIGN

Colorway

The colorscheme will be pallette colors. The images and design will be a mix of Neomorphism and SOFT UI if possible. Animations and images will be 3D and centered around technology.

INSPIRATIONS

COSMOS BLOCKCHAIN BERKELEY MESSARI

SCOPE

I plan to have 5 pages

  • Home
  • News
  • Watchlist
  • What is Blockchain?
  • About

EXTRAS

All pages will be hyperlinked.

There will be an AJAX sign in to customize user experience and display the users name atop each page.

Local storage will be utilized to store user info.
